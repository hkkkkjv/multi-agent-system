# Output Format — Форматы ответов агентов

## Planner: план дня

```
09:00 — Python (45 мин): Прочитать главу 5, решить 3 задачи на списки
09:45 — История (45 мин): Повторить даты ВОВ, сделать таймлайн
10:30 — Перерыв (15 мин)
10:45 — Python (45 мин): Написать функцию сортировки
```

Правила:
- Каждая строка: `ЧЧ:ММ — Тема (N мин): конкретная задача`
- Перерыв без задачи
- Самая срочная тема — первой

## Planner: микрозадача

```
Тема: Python
Время: 10 минут
Задача: Прочитай одну страницу про функции и сформулируй 1 вопрос
```

## Tutor: объяснение темы

```
## Тема: JOIN в SQL

**Простыми словами:** JOIN — это как склеить две таблицы по общей колонке.

**Пример из жизни:** Представь, у тебя есть список студентов и список оценок. 
JOIN соединит их так, чтобы ты видел: "Иван — 5, Мария — 4".

**Техническая суть:** 
- INNER JOIN: только совпадающие записи
- LEFT JOIN: все из левой + совпадающие из правой

**Запомни:** JOIN = "найди пару по ключу"
```

## Tutor: аналогии между темами

```
Тема 1: JOIN в SQL
Тема 2: Пересечение множеств (математика)

Аналогия 1: INNER JOIN = A ∩ B (только общие элементы)
Аналогия 2: LEFT JOIN = A + (A ∩ B) из B
Аналогия 3: CROSS JOIN = декартово произведение A × B
```
